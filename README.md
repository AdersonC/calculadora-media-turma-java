# Calculadora de Média da Turma em Java

Este projeto foi desenvolvido em Java com o objetivo de calcular a média das notas de uma turma.

O programa solicita ao usuário:

- A quantidade de alunos
- A quantidade de notas por aluno
- As notas de cada aluno

Após inserir os dados, o sistema calcula e exibe a média geral da turma.

## Tecnologias utilizadas

- Java
- IntelliJ IDEA
- Programação orientada a objetos (conceitos básicos)

## Conceitos aplicados

- Estruturas de repetição (for)
- Arrays bidimensionais
- Entrada de dados com Scanner
- Cálculo de média

## Como executar o projeto

1. Baixe ou clone o repositório
2. Compile o arquivo Java
3. Execute o programa

Exemplo:

javac exerciciosLP.java  
java exerciciosLP
